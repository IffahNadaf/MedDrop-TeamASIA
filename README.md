
# MedDrop Backend

This is the backend API for the MedDrop medicine donation application.

## Features

- User registration and login
- Add and view medicine donations
- Search for medicines
- Request medicines
- View notifications

## Tech Stack

- Node.js with Express
- Swagger UI for API documentation
